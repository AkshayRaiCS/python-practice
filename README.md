# Python Practice 

## What’s inside

### Number Guessing Game

My first real Python project.

The computer generates a number and you try to guess it. It helps in understanding how **loops, conditions, and user input** work together.

### Contact Book (not complete yet)

The program lets you save and retrieve contacts using a file, so the data actually persists between runs. While building this, I practiced:

* File handling
* Classes
* Error handling
* Organizing code across multiple files

### Practice Exercises

A collection of small programs I wrote while learning the fundamentals.

Topics include:

* loops
* string manipulation
* lists and dictionaries
* small logic problems
* APIs
* importing

## What I actually learned

The most valuable thing I learned was how to **read an error message, understand what went wrong, and fix it myself** instead of getting stuck.

That habit alone makes learning programming much easier.

## How to run the programs

Clone the repository:

```bash
git clone https://github.com/AkshayRaiCS/python-practice.git
cd python-practice
```

Run any file with Python:

```bash
python filename.py
```

Each program runs independently in the terminal.
